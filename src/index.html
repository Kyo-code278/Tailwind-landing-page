TailwindCSS

Crie sites modernos rapidamente

Um framework CSS utilitário repleto de classes como flex, pt-4, text-center e rotate-90 que podem ser combinadas para criar qualquer design, diretamente no seu código.

Index.html


<div class="flex items-center justify-center h-screen">
  <h1 class="text-4xl font-bold text-gray-900">
  Olá, Tailwind CSS!
  </h1>
</div>

Por que os desenvolvedores amam

Tailwind CSS fornece todos os blocos de construção necessários para criar sites bonitos e responsivos sem nunca sair do seu HTML.

Desenvolvimento ultrarrápido

Construa seus designs mais rápido com classes utilitárias que eliminam a troca de contexto entre HTML e CSS.

Completamente personalizável

Tailwind foi projetado para ser customizado. Altere cores, espaçamentos, tipografia e tudo mais com seu próprio sistema de design.

Responsivo por padrão

Crie interfaces totalmente responsivas com modificadores responsivos que facilitam adaptar seu design a qualquer tamanho de tela.

Ideal para componentes

Extraia padrões repetidos em componentes personalizados enquanto mantém o fluxo de trabalho utilitário que você adora.

Otimizado para produção

Remove automaticamente CSS não utilizado em builds de produção para performance ideal e tamanhos mínimos de arquivo.

Recursos modernos

Suporte para modo escuro, CSS Grid, animações e todos os recursos CSS mais recentes necessários para desenvolvimento web moderno.

Escreva menos, conquiste mais

Veja como o Tailwind CSS transforma a maneira como você escreve estilos, tornando seu código mais limpo e fácil de manter.

Index.html


<div class="flex items-center max-w-sm mx-auto p-6"
        bg-white rounded-lg shadow-xl>
  <div class="flex-shrink-0">
      <img class="h-12 w-12"src="/logo.svg" alt="Logo">
  </div>
  <div class="ml-6">
    <h4 class="text-xl text-gray-900">Nova mensagem!</h4>
    <p class="text-base text-gray-600">Você tem uma nova mensagem!</p>
  </div>
</div>
